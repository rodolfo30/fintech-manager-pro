# FinTech Manager Pro 🚀

Sistema corporativo de gestión automatizada de operaciones financieras bancarias, diseñado bajo una arquitectura limpia en capas y preparado para flujos de Integración Continua (CI).

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java 25 / JDK
* **Gestor de Dependencias:** Apache Maven
* **Testing & Mocking:** JUnit 5 y Mockito
* **Calidad de Código:** JaCoCo (Cobertura del 87.5%) y SonarQube (Métrica A)
* **Automatización CI/CD:** Jenkins (Pipeline declarativo via `Jenkinsfile`)

## 📦 Instrucciones de Construcción Local
Para compilar el proyecto y ejecutar la batería de pruebas unitarias de forma manual, ejecute:
